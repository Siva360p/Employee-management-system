# Employee Management Frontend

React + Vite interface for managing employees, including salary display and sortable hire date/salary columns.

## Requirements

- Node.js 20+
- npm 10+
- Backend running at `http://localhost:8080`

## Run

```bash
npm install
npm run dev
```

The app runs at `http://localhost:5173`.

To point the frontend at another API URL, create `.env`:

```bash
VITE_API_BASE_URL=http://localhost:8080/api
```
